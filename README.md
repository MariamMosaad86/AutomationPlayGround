##  Selenium Java-base UI Automation Framework

### Overview
This Framework is based on Selenium WebDrive with Java to automate UI scenarios

### Tech Stack
- Java
- Selenium WebDriver 
- TestNg
- Maven
- Allure Report

### Automation Framework Structure:
-This framework is structured as follows:
 - in Main Folder
  - Driver Factory
  - Browser Actions
  - Element Actions
  - Listeners
  - Pages
  - Utilities
